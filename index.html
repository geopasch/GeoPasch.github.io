This website contains a sample of data analysis and data visualization projects based on MySQL, Tableau or Microsoft Excel. The project ideas include an analysis of COVID-19 data for EU/EEA countries, cleaning of a FIFA 21 player attributes dataset, and a dashboard with information about modern Summer Olympics from 1896 until 2016.
